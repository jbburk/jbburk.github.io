White Space and Clean Design

Apple

apple.com

White Space is a fundamental part of a Clean Design. Despite the name "White Space" the color does not have to be White. Instead White Space refers to the color, texture, pattern or image that makes up the background. White Space allows for clear Visual Heirarchy to enhance a user's experience. Using White Space effectively allows the user to "breathe" or rest their eye between images and pragraphs.

Repetition

Next Door Pub & Pizzeria

nextdoorpub.com

Repetition is repeating a single element in a design many times. In this example Repetition is the box around each option. The Repetition in this page allows the user to expect that each box leads to a different choice. Most webpages use Repetition for easy user access. One example is having a companies logo in the top-left corner of the browser, followed by a navigation bar that starts with home. These common elements tie the whole internet together.

Visual Hierarchy

Marriott Hotels

marriott.com

Visual Hierarchy is using the size of text or images to signal what should be viewed first. For example, in this webpage we notice that below the image is the text "Travel with Peace of Mind". We notice this first because it the font is larger, bold, and is wider than the points below it. Therefore our eyes naturally arrive to the subsection title first and then proceed down the list.
